# ДЗ №10 — Завдання 2 (Монте-Карло)

## Що робив
Обчислив визначений інтеграл для **f(x) = x²** на відрізку **[0; 2]** методом Монте-Карло і порівняв результат:
- з аналітичним значенням;
- з `quad` з `scipy.integrate`.

## Метод Монте-Карло
Беремо `n` випадкових точок `x` на [a; b], рахуємо середнє значення `f(x)` і множимо на довжину відрізка:

I ≈ (b - a) * (1/n) * Σ f(x_i)

У коді: `n = 100000`.

## Перевірка
Аналітично:
∫0..2 x² dx = 8/3 ≈ 2.6666666667

Також пораховано через `quad(f, 0, 2)`.

## Висновок
Результат Монте-Карло близький до аналітичного та `quad`. Невелика різниця — це нормально, бо метод випадковий. Якщо збільшити `n`, похибка стане меншою.
